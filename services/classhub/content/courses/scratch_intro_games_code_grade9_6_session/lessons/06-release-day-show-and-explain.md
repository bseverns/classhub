**Theme:** Completion and explanation

## Objectives
- Finalize and submit release remix.
- Explain game loop using vocabulary.
- Write short reflection proving learning growth.

## Release checklist
- Controls work.
- Clear win and lose conditions.
- Score, lives, or timer is present and working.
- At least two feedback elements are present.

## Plan (60 minutes)
0-5 min: Entry routine.  
5-10 min: Mini-lesson on release checklist and prioritization.  
10-35 min: Final build sprint with checklist verification.  
35-50 min: Showcase rotation (partner swap or stations).  
50-55 min: Final submission.  
55-60 min: Closing statements on transfer:
- "You can read code."
- "You can change rules."

## Final reflection prompt (6-8 sentences)
- What did you change?
- What rule did you encode?
- What variable did you use?
- What feedback did you add?
- What would you improve next?

## Vocabulary focus
- Gameplay loop
- Event
- Condition
- Variable
- Feedback
